# Resoft API - Starter
Proyecto API REST (Node.js + Express + Sequelize + MySQL) para la Tienda Online Resoft.

## Pasos para ejecutar
1. Clonar el repositorio.
2. Crear un archivo `.env` con las variables del ejemplo.
3. `npm install`
4. Crear la base de datos y la tabla usando `sql/schema.sql`.
5. `npm run dev` (requiere nodemon) o `npm start`

## Endpoints principales
- GET /api/productos
- POST /api/productos
- PUT /api/productos/:id
- DELETE /api/productos/:id
